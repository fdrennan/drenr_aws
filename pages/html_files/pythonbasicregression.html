Individual basis functions¶

The individual basis functions have to be defined via a matrix approach (i.e., using a NumPy ndarray object). First, the case with monomials up to order 3.

In [60]:
matrix = np.zeros((3 + 1, len(x)))
matrix[3, :] = x ** 3
matrix[2, :] = x ** 2
matrix[1, :] = x
matrix[0, :] = 1

The sublibrary numpy.linalg provides the function lstsq to solve least-squares optimization problems like the one above.

In [84]:
reg = np.linalg.lstsq(matrix.T, f(x))[0]

def reg_func(a,x,y):
    f6 = a[6] * np.sqrt(y)
    f5 = a[5] * np.sin(x)
    f4 = a[4] * y ** 2
    f3 = a[3] * x ** 2
    f2 = a[2] * y
    f1 = a[1] * x
    f0 = a[0] * 1
    return(f6 + f5 + f4 + f3 + f2 + f1 + f0)
